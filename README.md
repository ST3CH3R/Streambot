# 🎮 StreamBot

<p align="center">
  <img src="https://raw.githubusercontent.com/ST3CH3R/Streambot/main/logo.png" width="200">
</p>

Ein moderner Streaming-Bot für Twitch und TikTok mit Chat-Commands, TTS, Soundboard und Stream-Benachrichtigungen.

---

## 📥 Download

➡️ **Neueste Version herunterladen:**

### 🚀 https://github.com/ST3CH3R/Streambot/releases/latest

---

## ✨ Features

### 🎥 Streaming
- Twitch Chat Integration
- TikTok Chat Integration
- Live Benachrichtigungen
- Multi-Plattform Unterstützung

### 🔊 TTS (Text-To-Speech)
- Chat vorlesen lassen
- Verschiedene Stimmen
- Lautstärke einstellbar

### ⚡ Commands
- Benutzerdefinierte Commands
- Sound Commands
- Eigene Antworten
- Moderation

### 🎵 Soundboard
- Eigene Sounds
- Sound-Auslöser per Chat
- Sofortige Wiedergabe

### ⚙️ Einstellungen
- Twitch Token Verwaltung
- TikTok Verbindung
- Benutzerfreundliche Oberfläche
- JSON Konfiguration

# 🛠 Installation

## Methode 1 (Empfohlen)

1. Öffne die Releases Seite
2. Lade die neueste Version herunter
3. Starte `StreamBot.Setup.exe`
4. Installation abschließen
5. StreamBot starten

---

## Twitch Einrichtung

1. Twitch Token generieren
2. Token einfügen
3. Kanalnamen eintragen
4. Verbinden

---

## TikTok Einrichtung

1. TikTok Live öffnen
2. TikTok Daten eintragen
3. Verbinden
4. Fertig
