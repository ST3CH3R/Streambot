STREAMBOT STABLE FIX

Diese Version behebt:
- Buttons reagieren wieder
- Sound hinzufügen speichert automatisch
- Sound löschen speichert automatisch
- Keine Standard-Sounds mehr
- Bot-Status zeigt Getrennt / Läuft / Fehler
- Einstellungen werden im Windows Benutzerordner gespeichert

Start:
npm install
npm start

TWITCH TOKEN:
Siehe Datei:
TWITCH_TOKEN_ANLEITUNG.txt

Kurz:
https://twitchtokengenerator.com/ öffnen,
"Bot Chat Token" wählen,
Token kopieren,
im StreamBot unter Twitch bei oauth:token eintragen.

EXE bauen:
npm run build

Danach:
dist\win-unpacked\StreamBot.exe

SOUNDS:
Der sounds Ordner liegt neben der StreamBot.exe.
